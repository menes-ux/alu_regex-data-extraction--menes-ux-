# ALU Regex Data Extraction - {YourUsername}

## Project Overview
This project is part of the first formative assignment for practicing **regular expressions (regex)** in JavaScript.  
The goal is to implement input validations for different types of data using regex patterns.

---

## Validations Implemented
The assignment requires implementing at least four of the following:

- **Email addresses**  
  Example: `user@example.com`, `firstname.lastname@company.co.uk`

- **Phone numbers**  
  Example: `(123) 456-7890`, `123-456-7890`, `123.456.7890`

- **URLs**  
  Example: `https://www.example.com`, `https://subdomain.example.org/page`

- **Credit card numbers**  
  Example: `1234 5678 9012 3456`, `1234-5678-9012-3456`

- **Time formats**  
  Example: `14:30` (24-hour), `2:30 PM` (12-hour)

- **HTML tags**  
  Example: `<p>`, `<div class="example">`, `<img src="image.jpg" alt="description">`

- **Hashtags**  
  Example: `#example`, `#ThisIsAHashtag`

- **Currency amounts**  
  Example: `$19.99`, `$1,234.56`
